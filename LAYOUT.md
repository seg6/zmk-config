## Layers
| Layer  | Access                   | Exit            |
|--------|--------------------------|-----------------|
| NAV    | left middle thumb        | release         |
| SYM    | right middle thumb       | release         |
| FUNC   | NAV + SYM                | release         |
| FKEYS  | NAV + T                  | NAV + T         |
| MOUSE  | FUNC + B                 | P               |
